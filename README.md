# calculatrice

Une calculatrice graphique avec historique, permettant d'effectuer des calculs mathématiques simples et avancés à l'aide d'une interface claire et intuitive.

## Aperçu de l'interface

<img width="448" height="330" alt="image" src="https://github.com/user-attachments/assets/9bdf1512-ff73-47ca-bc3a-077f4a68bff3" />


L'interface est divisé en deux partie 
  * Calculatrice : afficheur numérique et boutons d'opérations mathématiques
  * Historique :  affiche l'historique des calculs + bouton pour le reset

## Fonctionnalités

### Calculs pris en charge

* Addition (`+`)
* Soustraction (`-`)
* Multiplication (`*`)
* Division (`/`)
* Division entière (`//`)
* Modulo (`%`)
* Carré (`q`)
* Puissance (`^`)
* Factorielle (`!`)
* Racine carrée (`V`)
* Parenthèses (`(` `)`)

### Historique

* Affichage de l'historique des calculs effectués
* Bouton **Reset Historique** pour effacer l'historique
* Message *"Historique vide"* lorsqu'aucun calcul n'est enregistré

### Interface utilisateur

* Bouton **C** pour effacer la saisie en cours
* Bouton **=** pour afficher le résultat
* Saisie des opérations via les boutons de l'interface

## Technologies utilisées

* Langage : selon l'implémentation du projet
* Interface graphique : application avec interface graphique

## Améliorations possibles

* Support de la saisie clavier
* Ajout d'un mode scientifique avancé
* Sauvegarde automatique de l'historique
* Ajout de thèmes (clair / sombre)
